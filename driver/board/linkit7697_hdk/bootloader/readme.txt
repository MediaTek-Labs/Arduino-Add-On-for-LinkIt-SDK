Bootloader main module usage guide

Brief:          This module only for bootloader, it is bootloader implementation.
Usage:          GCC: Include the module with "include $(SOURCE_DIR)/driver/board/mt76x7_hdk/bootloader/module.mk" in your GCC bootloader project Makefile .
                KEIL: Drag the driver/board/mt76x7_hdk/bootloader/src folder to your bootloader project . Add driver/board/mt76x7_hdk/bootloader/src/mt7687 to INCLUDE_PATH.
                IAR: Drag the driver/board/mt76x7_hdk/bootloader/src folder to your bootloader project . Add driver/board/mt76x7_hdk/bootloader/src/mt7687 to INCLUDE_PATH.
